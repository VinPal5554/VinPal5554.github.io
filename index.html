Notable Projects


Malware Prediction Web Application

Final year project that explores clustering to predict new strains of malware. It uses techniques such as: Hierarchical, K-Means, DBSCAN, and Consensus Clustering and uses the Weighted Kappa metric to evaluate them. Originally Made with R, and post-university set up a GUI via Shiny.

Languages/Tech: R, Shiny

Weather App

A simple weather forecasting application built using Vue.js for the frontend and Flask for the backend. This app fetches real-time weather data from the OpenWeatherMap API and displays it to the user. Utilises HTML and CSS for a simple but clean UI for the user.

Languages/Tech: Python, Flask, Vue.js, HTML, CSS, JavaScript

PokeVin - Card Price Tracker

Web-scraping tool that fetches Pokemon card prices. It uses Playwright to scrape import metrics such as price/title/source. These metrics are stored in a SQLite database via Django. More recently have integrated this into a Discord bot for better user experience.

Languages/Tech: Python, Django, Playwright, Discord.py

FoodHub: Food Donation Application

Foodbank app that uses points to incentivise donations by allowing points to be redeemed for vouchers. Made with Spring Boot, JavaScript, Tomcat, and Microsoft SQL Server. Utilises Spring Security for authentication and Spring Admin for a potential admin user.

Languages/Tech: Java, Spring Boot, TomCat, SQL Server, HTML, CSS, JavaScript

Notebook App

A full-stack note-taking application built with TypeScript, Express.js, Tailwind CSS, and MongoDB. It features a frontend built in plain HTML/JS and allows users to add and view, and delete notes stored in a MongoDB database. Users are also able to create custom tags/categories.

Languages/Tech: TypeScript, Express.js, Tailwind, MongoDB

Job Application Tracker

Web-based job application tracker built using PHP, MariaDB/MySQL, and HTML/CSS. It allows the user to enter, edit, and adjust applications, send reminders to the user about upcoming interviews/assessments, and sends follow-up emails to employers. Makes use of phpMyAdmin to engage with database.

Languages/Tech: PHP, HTML, CSS, MySQL

Photo Editor App

Web-based photo editor built with React.js, HTML5 Canvas API, and Tailwind CSS. This project allows users to upload images and perform basic image manipulation tasks like rotating, flipping, and adjusting image size. In addition to this I have set up basic filters and a crop tool.

Languages/Tech: React.js, HTML5 Canvas API, Tailwind CSS

Infernal Arena: Roguelike Arena Battler

A personal ongoing game project made using the Unity Engine. It is a 2d pixel-art roguelike where the player battles in an arena against increasingly stronger enemies. Utilises interesting game mechanics such as procedurally generated enemies, a shop system, and boss fights.

Languages/Tech: C Sharp, Unity

Console File Explorer

C++ program that allows the user to navigate through directories via the terminal/command prompt. Utilises features similar to a command prompt such as breadcrumb navigation, and colour coding for ease of use. I plan to add features such as file manipulation, sorting/filtering, and scripting functionality.

Languages/Tech: C++
